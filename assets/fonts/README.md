# Fonts

Replace `GeneralSans-Variable.ttf` with the real **General Sans** variable font:

1. Go to https://www.fontshare.com/fonts/general-sans
2. Download the family (free).
3. Copy the variable TTF here as `GeneralSans-Variable.ttf` (overwrite the placeholder).
4. Reload the app (`npx expo start -c`).

Until you do, the app falls back to the system font automatically.
